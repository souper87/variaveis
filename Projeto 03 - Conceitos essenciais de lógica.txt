# Projeto 03 - Conceitos essenciais de lógica

## Instruções do projeto

Em uma inscrição, o usuário informou os seguintes dados:

- Nome: José Almeida da Silva
- CPF: 12345678900
- RG: 9517530
- Altura: 1,78
- Endereço: Rua A, 380 – Centro – Recife/PE

No algoritmo, descreva como será:
1. o identificador das variáveis;
2. a declaração das variáveis com seus respectivos identificadores e tipos de dado;
3. a utilização do comando de atribuição, apresentando identificador e dado.

## Criação do algoritmo

### 1. identificador das variáveis

Nome = constante do tipo caractere
RG e CPF = constantes do tipo real
Altura = variável do tipo real
Endreço = variável do tipo caractere

### 2. Declaração das variáveis

const Nome = "josé-almeida-da-silva"
const Cpf = 12345678900
const Rg = 9517530
var Altura = 1,78
var Endereço = "rua-a-380-centro-recife-pe"

### 3. Comando de atribuição

const nome <-"josé-almeida-da-silva"
const cpf <-12345678900
const rg <-9517530
